# Managers Directory

## Description: External tools required for mod management

This directory contains external tools required for Wildflover to function properly.

## Required Files

| File | Description | Source |
|------|-------------|--------|
| `cslol-diag.exe` | Diagnostic tool | [cslol-manager](https://github.com/LeagueToolkit/cslol-manager) |
| `cslol-dll.dll` | Core library | [cslol-manager](https://github.com/LeagueToolkit/cslol-manager) |
| `mod-tools.exe` | Mod management | [cslol-manager](https://github.com/LeagueToolkit/cslol-manager) |
| `wad-extract.exe` | WAD extraction | [cslol-manager](https://github.com/LeagueToolkit/cslol-manager) |
| `wad-make.exe` | WAD creation | [cslol-manager](https://github.com/LeagueToolkit/cslol-manager) |
| `hashes.game.txt` | Game file hashes | [cslol-manager](https://github.com/LeagueToolkit/cslol-manager) |

## Installation

1. Download the latest release from [cslol-manager releases](https://github.com/LeagueToolkit/cslol-manager/releases)
2. Extract the following files to this directory:
   - `cslol-diag.exe`
   - `cslol-dll.dll`
   - `mod-tools.exe`
   - `wad-extract.exe`
   - `wad-make.exe`
   - `hashes.game.txt`

## Batch Scripts

The following batch scripts are included for convenience:

- `wad-extract-multi.bat` - Extract multiple WAD files
- `wad-make-multi.bat` - Create multiple WAD files
- `wxy-extract-multi.bat` - Extract WXY files

## Note

These binary files are not included in the repository due to licensing.
Please download them from the official source.
